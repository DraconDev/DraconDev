# 🛸 SOVEREIGN SIMULATION: Junk-Runner

**"Amazing Space through Pure Math."**

`Junk-Runner` is a real-time tactical deck-building space exploration roguelike. It serves as a **High-Performance Reference Implementation** for the Bevy game engine.

## 💎 The Technical Flex: 100% Procedural

Most games are built with gigabytes of textures and meshes. `Junk-Runner` is different:
- **Zero Static Assets:** Every ship, star, nebula, and UI element is rendered using **Bevy Gizmos** and procedural shaders.
- **Bevy 0.18:** Built on the bleeding edge of the engine's development.
- **Performance:** 60FPS high-fidelity space combat with a minimal memory footprint.

## 🛠️ Standout Features

- **Singularity Visuals:** The "Red Dot" enemy is a masterclass in volumetric glow and orbital particle simulation using pure math.
- **BRICKS UI Framework:** A custom, lightweight UI system designed for high-intensity game interfaces that Bevy's default UI struggles to handle.
- **Tactical Compass:** A high-fidelity Teal (Cyan) navigational aid that provides 3D orientation in a void background.

## 🎮 Why it Matters

`Junk-Runner` is the proof-of-concept for the **Dracon Engineering Philosophy**:
- **Math over Bloat:** Use the GPU for logic, not just storage.
- **Identity over Fidelity:** Create a unique "Neural Terminal" aesthetic that stands out more than generic photorealistic assets.
