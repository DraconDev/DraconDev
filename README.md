# DraconDev

Rust-first tools for building fast, reliable software.

| Project | Language | What it is |
|:---|:---:|:---|
| **[azumi](https://github.com/DraconDev/azumi)** | Rust | Compile-time validated web framework. Zero hydration, ~3KB runtime. |
| **[dracon-terminal-engine](https://github.com/DraconDev/dracon-terminal-engine)** | Rust | TUI app framework. 35 widgets, 15 themes, z-indexed compositing. [crates.io](https://crates.io/crates/dracon-terminal-engine) |
| **[tiles](https://github.com/DraconDev/tiles)** | Rust | Terminal file manager with editor, git integration, SSH browsing. |
| **[dracon-libs](https://github.com/DraconDev/dracon-libs)** | Rust | Shared libraries: git, system, media, memory, AI routing. |
| **[ai-vid-editor](https://github.com/DraconDev/ai-vid-editor)** | Rust | Automated video editing. Silence removal, auto-reframe, audio enhancement. |
| **[api-debugger](https://github.com/DraconDev/api-debugger)** | TypeScript | Chrome extension. Capture, replay, modify HTTP requests. No account required. |
| **[opencode-auto-force-resume](https://github.com/DraconDev/opencode-auto-force-resume)** | TypeScript | OpenCode plugin. Detects and recovers stalled AI sessions. |
| **[kittentts-showcase](https://github.com/DraconDev/kittentts-showcase)** | Python | Text-to-speech showcase. |
