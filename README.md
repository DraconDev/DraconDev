## Ecosystem

### Web & UI

| | | |
|:---|:---|:---|
| **[azumi](https://github.com/DraconDev/azumi)** | Compile-time validated web framework for Rust. Catches CSS typos and HTML errors before deployment. | `rust` `web` `compiler` |
| **[dracon-rust-ui](https://github.com/DraconDev/dracon-rust-ui)** *(private)* | ECS-based GPU UI engine. Vello rendering, 10 themes, 44 interactive showcases. | `rust` `gpu` `ui` |

### Terminal & CLI

| | | |
|:---|:---|:---|
| **[dracon-terminal-engine](https://github.com/DraconDev/dracon-terminal-engine)** | TUI app framework with 35 widgets, 15 themes, z-indexed compositing. Ships as a single binary. | `rust` `tui` `framework` |
| **[tiles](https://github.com/DraconDev/tiles)** | Terminal file manager. Dual-pane navigation, text editor, git integration, SSH browsing. | `rust` `terminal` `file-manager` |

### Developer Tools

| | | |
|:---|:---|:---|
| **[dracon-code](https://github.com/DraconDev/dracon-code)** *(private)* | Autonomous engineering runtime. Blueprint-first execution with dark GUI and CLI. Fleet management, gate verification, recovery. | `rust` `ai` `automation` |
| **[dracon-libs](https://github.com/DraconDev/dracon-libs)** | Shared Rust libraries: git operations, system monitoring, media processing, AI routing. | `rust` `libraries` |
| **[api-debugger](https://github.com/DraconDev/api-debugger)** | Chrome extension for debugging HTTP requests. Capture, replay, modify. No account required. | `typescript` `chrome-extension` |
