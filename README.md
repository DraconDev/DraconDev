# DraconDev

Building software that doesn't break in production.

## Flagship

| | |
|:---|:---|
| **[azumi](https://github.com/DraconDev/azumi)** | Compile-time validated web framework for Rust. Catches CSS typos and HTML errors before deployment. Zero hydration, ~3KB runtime. |
| **[dracon-terminal-engine](https://github.com/DraconDev/dracon-terminal-engine)** | TUI app framework with 35 widgets, 15 themes, and z-indexed compositing. Ships as a single binary. [crates.io](https://crates.io/crates/dracon-terminal-engine) |

## Tools

| | |
|:---|:---|
| **[tiles](https://github.com/DraconDev/tiles)** | Terminal file manager. Dual-pane navigation, text editor, git integration, SSH browsing. |
| **[dracon-libs](https://github.com/DraconDev/dracon-libs)** | Shared Rust libraries: git operations, system monitoring, media processing, AI routing. |
| **[ai-vid-editor](https://github.com/DraconDev/ai-vid-editor)** | Automated video editing. Silence removal, auto-reframe, audio enhancement, watch folders. |
| **[api-debugger](https://github.com/DraconDev/api-debugger)** | Chrome extension for debugging HTTP requests. Capture, replay, modify. No account required. |
