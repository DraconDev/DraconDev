# DraconDev

> Rust-first tools. Compile-time guarantees.

## Ecosystem

### Web & UI

| | |
|:---|:---|
| **[azumi](https://github.com/DraconDev/azumi)** | Compile-time validated web framework. Catches CSS typos and HTML errors before deployment. |
| **[dracon-rust-ui](https://github.com/DraconDev/dracon-rust-ui)** *(private)* | ECS-based GPU UI engine. Vello rendering, 10 themes, 44 interactive showcases. |

### Terminal & CLI

| | |
|:---|:---|
| **[dracon-terminal-engine](https://github.com/DraconDev/dracon-terminal-engine)** | TUI app framework with 35 widgets, 15 themes, z-indexed compositing. Ships as a single binary. [crates.io](https://crates.io/crates/dracon-terminal-engine) |
| **[tiles](https://github.com/DraconDev/tiles)** | Terminal file manager. Dual-pane navigation, text editor, git integration, SSH browsing. |

### Developer Tools

| | |
|:---|:---|
| **[dracon-code](https://github.com/DraconDev/dracon-code)** *(private)* | Autonomous engineering runtime. Blueprint-first execution with dark GUI and CLI. Fleet management, gate verification, recovery. |
| **[dracon-libs](https://github.com/DraconDev/dracon-libs)** | Shared Rust libraries: git operations, system monitoring, media processing, AI routing. |
| **[api-debugger](https://github.com/DraconDev/api-debugger)** | Chrome extension for debugging HTTP requests. Capture, replay, modify. No account required. |

### Content Creation

| | |
|:---|:---|
| **[ai-vid-editor](https://github.com/DraconDev/ai-vid-editor)** | Automated video editing. Silence removal, auto-reframe, audio enhancement, watch folders. |
| **[kittentts-showcase](https://github.com/DraconDev/kittentts-showcase)** | Real-time text-to-speech demo using the Kitten TTS engine. |

### OpenCode Plugins

| | |
|:---|:---|
| **[opencode-auto-review-completed-todos](https://github.com/DraconDev/opencode-auto-review-completed-todos)** | Auto-reviews completed sessions. Tracks todos, detects completion, summarizes work. |
| **[opencode-auto-force-resume](https://github.com/DraconDev/opencode-auto-force-resume)** | Recovers stalled AI sessions automatically. Smart stall detection with retry limits. |
