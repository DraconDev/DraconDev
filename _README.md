## Rust Engines & Libraries

Core frameworks and reusable components.

* **[azumi](https://github.com/DraconDev/azumi)** — Live SSR for Rust, builds on Axum. ~10KB gzipped, 41 widgets.
* **[terma](https://github.com/DraconDev/terma)** — Next-gen terminal compositor engine. Z-indexed layers, TrueColor, SGR mouse, drop-in Ratatui support.
* **[terminal-engine](https://github.com/DraconDev/dracon-terminal-engine)** — Terminal app framework. App, widgets, compositor, 15 built-in themes.
* **[dracon-libs](https://github.com/DraconDev/dracon-libs)** — Vertical Rust libraries — git, terminal, system, media, memory.
* **[agave](https://github.com/DraconDev/ai-gui-auto-video-editor)** — AI video editor. GUI + CLI, silence cutting, audio enhancement, scene detection, auto-reframe.
