# Dracon Ecosystem

Autonomous engineering runtime. Rust core with AI agents, TUI frameworks, and open tooling for developers who ship.

---

## 🔌 OpenCode Plugins

Tiny tools that make AI coding sessions less painful.

* **[opencode-auto-review-completed-todos](https://github.com/DraconDev/opencode-auto-review-completed-todos)** — Detects when all session TODOs are done, triggers review
* **[opencode-auto-continue](https://github.com/DraconDev/opencode-auto-continue)** — Stall recovery, todo nudging, session analysis
* **[opencode-auto-force-resume](https://github.com/DraconDev/opencode-auto-force-resume)** — Detects and recovers stalled AI sessions
* **[pi-auto-review](https://github.com/DraconDev/pi-auto-review)** — Scans projects for problems, writes TODO.md, auto-fixes

## 🛠️ Developer Tools

CLI tools and editor extensions.

* **[obs-wayland-hotkey](https://github.com/DraconDev/obs-wayland-hotkey)** — 8★ Rust daemon for OBS Studio hotkeys on Wayland/X11
* **[git-ai-committer](https://github.com/DraconDev/git-ai-committer)** — 6★ VS Code extension, auto-commits with AI messages on natural pauses
* **[git-seal](https://github.com/DraconDev/git-seal)** — Transparent encryption filter for Git
* **[youtube-video-uploader](https://github.com/DraconDev/youtube-video-uploader)** — Rust CLI for YouTube Data API v3, resumable uploads
* **[respec-spec-reconciler](https://github.com/DraconDev/respec-spec-reconciler)** — Spec reconciler for AI agents, reads SPEC.md, loops until all invariants pass
* **[css-peek-pro](https://github.com/DraconDev/css-peek-pro)** — VSCode extension, smart CSS scoping for large projects
* **[api-debugger](https://github.com/DraconDev/api-debugger)** — Chrome extension, captures/replays HTTP requests, imports from Postman/Insomnia

Core frameworks and reusable components.

* **[azumi-live-ssr-framework](https://github.com/DraconDev/azumi-live-ssr-framework)** — Live SSR for Rust, builds on Axum, ~10KB gzipped
* **[terma](https://github.com/DraconDev/terma)** — 1★ Next-gen terminal compositor engine, Z-indexed layers, TrueColor, SGR mouse, drop-in Ratatui support
* **[dracon-terminal-engine](https://github.com/DraconDev/dracon-terminal-engine)** — Terminal app framework, widgets, compositor, 15 built-in themes
* **[dracon-libs](https://github.com/DraconDev/dracon-libs)** — 1★ Vertical Rust libraries, git, terminal, system, media, memory
* **[ai-gui-auto-video-editor](https://github.com/DraconDev/ai-gui-auto-video-editor)** — AGAVE, GUI + CLI for automated video editing, silence cutting, audio enhancement, scene detection, auto-reframe
